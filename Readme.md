# 本研究計畫程式之使用說明 (Machine Learning)

本研究計畫之相關程式碼與資料，主要可分為 **Paper、Project** 以及 **Train Data** 三大部分。以下針對各資料夾之內容與用途進行說明。

---

## 1. Paper

此資料夾主要收錄本研究計畫在**撰寫論文期間所使用之程式碼與相關資料**，並依功能分為以下三個子資料夾：

### (1) main code
本資料夾包含研究中使用之主要模型訓練程式碼，主要應用 **五種機器學習模型** 進行蛋白質（Protein）含量預測。  
使用者可透過修改 `PARAMS` 參數，自行調整光譜波長範圍或切換不同模型設定。

其中 `data` 子資料夾存放程式執行後所整理之數據，以及相關分析結果與繪製之圖表。

### (2) matlab
此資料夾收錄論文中繪製重要圖表所使用之 MATLAB 程式碼，包含以下內容：

- 光譜原始數據曲線圖（`Imread_Spectrum`）
- MCS 與 Mahalanobis Distance（MD）異常值剔除圖表（`MCS-MD`）
- 各模型預測結果之散佈圖（`Scatter`）

### (3) process
此資料夾包含論文中所使用之技術比較程式碼，主要分為：

- 特徵提取方法比較（`feature_extraction (protein)`）
- 光譜數據前處理方法比較（`spectrum_process (protein)`）

各子資料夾中的 `data` 資料夾為程式執行後整理之結果數據，供後續分析與比較表格製作使用。

---

## 2. Project

此資料夾為**本研究計畫實際應用之完整系統程式與相關資料**，並依功能分為以下子資料夾：

### (1) main code
包含本研究主要訓練與預測之核心程式碼，主要應用 **四種模型** 進行下列成分之含量預測：

- 粗灰質（Ash）
- 脂肪（Fat）
- 水分（Moisture）
- 蛋白質（Protein）

同樣可透過修改 `PARAMS` 參數，調整光譜波長範圍或模型設定。

### (2) process
此資料夾包含研究中使用之技術比較程式碼，主要內容包括：

- 特徵提取方法比較（`feature_extraction (protein)`）
- 光譜數據前處理方法比較（`spectrum_process (protein)`）

各子資料夾中的 `data` 為程式執行後整理之結果數據，方便進行後續分析與成果比較。

### (3) Scatter
此資料夾收錄本研究中各模型預測結果之散佈圖（Scatter plot）繪製程式碼。

---

## 3. Train Data

此資料夾包含本研究進行模型訓練與分析時所使用之原始與處理後數據，主要分為以下幾類：

- `trad_data`：傳統化學量測數據  
- `nir_data`：近紅外線（NIR）光譜數據

此外，亦包含農業部原始提供之光譜資料（`.nir` 檔）及數據視覺化用之 `.cal` 檔。  
為利後續研究與程式應用，已使用 **SpectraGryph** 軟體將 `.nir` 檔轉換為 `nir_data.csv`。

`pre` 資料夾中則存放尚未整理之原始數據檔案，若造成使用不便，敬請見諒。
